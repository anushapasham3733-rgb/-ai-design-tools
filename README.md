# AI Design Tools

AI Design Tools is an open-source collection of prompts, templates, and utilities for AI-assisted design and digital product creation.

## Features

- AI image generation prompts
- Etsy SEO templates
- Digital product workflow examples
- Prompt engineering resources
- Open-source documentation

## Project Goals

This project aims to help creators, designers, and developers build better AI-assisted workflows for digital products.
